Welcome to Cosmetics Store Management System

Makeup lovers often want to find a single source of reliable data about their favourite makeup products. Thus, we’ve created a rest API to provide its users with the ability to search for make up products on basis of brands, product names and prices.

This applicaion is implemented as part of cloud computing project by Benita Ashley, Jagruthi Kiran, Komali, Prithvi, Prajwal.
